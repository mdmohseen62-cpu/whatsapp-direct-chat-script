# Direct WhatsApp Chat Script

A simple, lightweight JavaScript snippet to generate direct `wa.me` API URLs. This allows users to open a WhatsApp chat without saving the phone number to their contacts.

## Usage
Simply pass the country code and the phone number into the `openWhatsApp()` function. The script automatically strips out invalid characters and triggers the redirect.
